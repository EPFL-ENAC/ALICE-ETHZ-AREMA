## Zustände

Jedes Dokument (Ressource, Baumaterial usw.) kann einen der folgenden Zustände haben:

| Zustand                | Beschreibung                                                                                                      |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **Entwurf**            | Das Dokument wird hinzugefügt oder bearbeitet und ist noch nicht zur Überprüfung bereit.                          |
| **In Überprüfung**     | Das Dokument ist zur Überprüfung durch eine/n Prüfer/in bereit.                                                   |
| **Zu veröffentlichen** | Das Dokument wurde von einer/einem Prüfer/in überprüft und zur Veröffentlichung freigegeben.                      |
| **Zu depublizieren**   | Das Dokument ist derzeit veröffentlicht und muss von einem Administrator depubliziert werden.                     |
| **Zu löschen**         | Das Dokument ist von einem Administrator zur Löschung markiert.                                                   |
| **Gesperrt**           | Das Dokument ist von einem Administrator gesperrt und kann nicht bearbeitet, (de)publiziert oder gelöscht werden. |

## Rollen

Der Publikationsfluss umfasst verschiedene Rollen, jede mit spezifischen Berechtigungen:

| Rolle             | Berechtigungen                                                                                      |
| ----------------- | --------------------------------------------------------------------------------------------------- |
| **Beitragender**  | Kann eigene Dokumente hinzufügen und bearbeiten.                                                    |
| **Prüfer/in**     | Kann alle Dokumente hinzufügen und bearbeiten, aber nicht (de)publizieren oder löschen.             |
| **Administrator** | Kann alle Operationen durchführen, einschließlich der (De)publizierung und Löschung von Dokumenten. |
